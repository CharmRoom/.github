### 참방은 토이프로젝트로 수영 커뮤니티 참방을 만들 목표로 만들어진 팀 입니다.
<br>

## 팀원 소개

참방은 2명의 팀원으로 구성한 작은 팀 입니다.<br>
팀원 수가 적은 만큼 모든 과정은 두 팀원이 함께 진행하였습니다.
